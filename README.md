# lx-landing
开源单页精美广告源码 纯静态页面

## 项目简介
这是一个个人广告展示页面，主要用于展示"黎轩一站式智能加速与安全防护"服务的广告页面。

### 项目特点
- 响应式设计 : 支持桌面端、平板和移动端自适应布局
- 视觉设计 : 采用渐变背景和卡片式布局，具有现代感
- 服务展示 : 展示6个核心服务模块（合作方式、优势、移动被墙、CC防御、DDoS防护、团队实力）
- 联系方式 : 集成Telegram联系方式链接
- 数据统计 : 集成51.LA统计代码
### 技术栈
- 纯HTML5 + CSS3
- 无需JavaScript框架
- 响应式布局（Flexbox）
- 媒体查询适配不同设备
